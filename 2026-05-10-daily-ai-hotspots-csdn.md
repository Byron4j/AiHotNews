## 二、热点详解

## # 1. 浙大校友用AI破32年拉姆齐数下界：AI for Math 的新边界

**核心事件**：浙大校友团队利用AI方法，突破了32年未变的拉姆齐数（Ramsey number）下界。拉姆齐数是组合数学的核心难题——R(5,5)至今未解，Paul Erdős曾以"外星人入侵"比喻其难度。据量子位报道，该团队通过AI驱动的搜索和推理技术，成功发现了更大规模的拉姆齐图构造，从而提升了下界。

**为什么重要**：

- 此前AI在数学领域的应用主要为"解题"（如Google AI联合数学家解群论问题、AlphaProof拿IMO银牌），而拉姆齐数下界的突破属于"发现新数学事实"——难度更高
- 这标志着AI for Math从"助理解题"进入"独立发现"的新阶段
- 与Google Co-Mathematician（5月9日发布）形成呼应——本周两个独立团队同时展示了AI在纯数学研究中的应用潜力

## # 2. 高德ABot空间智能夺冠：具身AI的中国路线

**核心事件**：高德ABot体系模型在AGIBot全球挑战赛中以0.829总成绩夺冠。该赛事聚焦空间智能（Spatial Intelligence）与具身智能（Embodied AI）的结合——要求AI系统在三维空间中完成导航、识别、交互等复合任务。

**技术信号**：Jim Fan在5月9日红杉演讲中宣布"VLA已死，WAM当立"，高德ABot的夺冠验证了空间智能从实验室走向评测化的趋势。中国团队在具身智能赛道正在积累工程实践和评测数据。

## # 5. OpenAI发布Codex Chrome扩展：Agent从终端走进浏览器

**核心事件**：OpenAI为Codex发布Chrome浏览器扩展。该扩展允许Codex在用户已登录的网站和应用中完成工作，在"任务专属"标签组中运行，不干扰用户正在使用的标签页。

**Agent操作域扩展路线**：
- Claude Code → 终端操作
- Codex + Chrome Extension → 浏览器操作
- OpenClaw → 网页操控
- Jim Fan EgoScale → 物理世界操作

谁先控制更多操作域，谁就拥有更强的Agent经济护城河。
